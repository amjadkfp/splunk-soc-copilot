## рҹҺҜ Why SOC Copilot?

<div align="center">

### *Transform raw Windows Security Event Logs into actionable security intelligence.*

**Built for SOC analysts, cybersecurity learners, and security teams.**

</div>

Security analysts often face **thousands** of security events daily. Understanding whether an event represents normal activity or a potential attack requires significant time and expertise.

> рҹ’Ў **SOC Copilot accelerates investigations by automatically handling the heavy lifting so analysts can focus on what matters most.**

<div align="center">

### вҡЎ Accelerated Investigation Pipeline

</div>

| Step | Action | Outcome |
|:---:|:---|:---|
| рҹҺҜ | **Prioritize** suspicious events using severity scoring | Risk-based triage in seconds |
| рҹӨ– | **Explain** security events in plain English | No more decoding raw Event IDs |
| рҹ—әпёҸ | **Map** attacker behavior to MITRE ATT&CK | Adversary context instantly |
| рҹ”Қ | **Detect** attack patterns across multiple events | Catch chains, not just singles |
| рҹ“„ | **Generate** investigation and remediation guidance | Actionable checklists on demand |
| рҹ”— | **Forward** enriched findings to Splunk | Centralized SIEM correlation |

## рҹ“Ұ Installation & Running (Windows 11)

<div align="center">

### рҹӣ пёҸ *Get up and running in 5 minutes*

</div>

<br/>

### рҹ“Ӣ Prerequisites

> - вң… Python 3.10 or higher installed
> - вң… Internet connection *(optional, for AI-assisted analysis)*
> - вң… Git *(optional, for cloning the repository)*

<br/>

### рҹ”№ Step 1 вҖ” Clone or Download

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/soc-copilot.git
cd soc-copilot

# Or download the ZIP file and extract it
```

<br/>

### рҹ”№ Step 2 вҖ” Open Terminal

```bash
# Press Win + R, type cmd, press Enter
cd path\to\splunk-soc-copilot

# Or right-click the folder in Explorer вҶ’ "Open in Terminal"
```

<br/>

### рҹ”№ Step 3 вҖ” Create a Virtual Environment

```bash
# Create venv
python -m venv venv

# Activate it:
```

<details>
<summary><b>рҹҹҰ Command Prompt (CMD)</b></summary>

```bash
venv\Scripts\activate.bat
```

</details>

<details>
<summary><b>рҹҹҰ PowerShell</b></summary>

```bash
.\venv\Scripts\Activate.ps1
```

</details>

> рҹ’Ў You should see `(venv)` appear in your terminal prompt.

<br/>

### рҹ”№ Step 4 вҖ” Install Dependencies

```bash
pip install -r requirements.txt
```

<br/>

### рҹ”№ Step 5 вҖ” Run SOC Copilot

<div align="center">

| рҹҡҖ Option | рҹ“қ Method | вӯҗ Recommended |
|:---:|:---|:---:|
| **A** | Double-click `start_soc_copilot.bat` | вӯҗ **Yes** |
| **B** | Run `python app.py` from terminal | вң… Yes |

</div>

<br/>

**Option A вҖ” One-Click Launcher (Recommended)**
> Double-click: `start_soc_copilot.bat`

**Option B вҖ” Run from Terminal**
```bash
python app.py
```

<br/>

### рҹ”№ Step 6 вҖ” Open in Browser

<div align="center">

### рҹҢҗ Navigate to: **`http://127.0.0.1:5000`**

> рҹ’Ў Click **"Try Sample Logs"** to immediately see a full demo with a simulated attack scenario.

</div>

<br/>

### рҹӨ– Optional AI-Assisted Analysis

> To enable AI-assisted analysis, configure your API key before starting the application.

<details>
<summary><b>рҹҹҰ Command Prompt</b></summary>

```bash
set ANTHROPIC_API_KEY=your-api-key
```

</details>

<details>
<summary><b>рҹҹҰ PowerShell</b></summary>

```bash
$env:ANTHROPIC_API_KEY="your-api-key"
```
